# pxie

> [!WARNING]
> This library is still under early development.

Pxie aims to be a simple software for pixel art.

Roadmap:
- [ ] Draw a grid on screen.
- [ ] Draw a color picker on screen.
- [ ] Paint grid cells with the selected color on user click.
- [ ] Save art to PNG.
- [ ] Save and open the art state to a .pxie file.
